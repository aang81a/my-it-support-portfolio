## Purpose

This mini-report documents a simulated first-level IT support incident involving a OneDrive sync issue.

The goal is to show how a support technician can document the issue, user impact, troubleshooting steps, resolution, verification, and prevention notes in a clear and professional format.

## User Reported Issue

The user reported that OneDrive was not syncing files correctly.

The user noticed that changes made to a file were not appearing in OneDrive online, and the OneDrive icon showed a sync warning.

---

## Symptoms Observed

- OneDrive showed a sync error.
- One file remained in sync pending status.
- The user was signed in to Microsoft 365.
- Other internet services were working.
- The affected file was not updated in OneDrive online.

---

## Initial Questions Asked

- When did the sync issue start?
- Is the issue affecting one file or multiple files?
- Are you signed in with your work or school Microsoft 365 account?
- Does the file appear in OneDrive online?
- Do you see an error message or warning icon?
- Did the file name contain special characters?
- Was the file open in another application?

---

## Troubleshooting Steps

### 1. Checked OneDrive Status

The OneDrive cloud icon was checked in the taskbar notification area.

Result:

- OneDrive was running.
- The user was signed in.
- A sync warning was visible.

---

### 2. Checked Internet Connection

The internet connection was verified.

Result:

- Web browsing worked.
- Microsoft 365 sign-in worked.
- The issue was not caused by a general network outage.

---

### 3. Checked File Name

The affected file name was reviewed for unsupported characters.

Result:

- No unsupported characters were found.
- The file name did not need to be changed.

---

### 4. Checked OneDrive Online

The user opened OneDrive online and searched for the affected file.

Result:

- The file existed online.
- The latest local changes were not visible online.

---

### 5. Restarted OneDrive

OneDrive was closed and opened again.

Steps:

1. Clicked the OneDrive cloud icon.
2. Opened **Help & Settings**.
3. Selected **Quit OneDrive**.
4. Reopened OneDrive from the Start menu.
5. Waited for sync to restart.

Result:

- OneDrive restarted successfully.
- The sync warning remained.

---

### 6. Restarted the Computer

The user saved work and restarted the computer.

Result:

- OneDrive started again after sign-in.
- The affected file started syncing.
- The warning icon disappeared after a few minutes.

---

## Root Cause

The likely root cause was a temporary OneDrive sync process issue on the local computer.

The issue was resolved after restarting the computer and allowing OneDrive to restart the sync process.

---

## Resolution

The issue was resolved by:

- Confirming the user was signed in to OneDrive.
- Checking that the internet connection worked.
- Reviewing the affected file name.
- Comparing the local file with OneDrive online.
- Restarting OneDrive.
- Restarting the computer.
- Verifying that the file synced successfully.

---

## Verification

The resolution was verified by checking that:

- The OneDrive icon no longer showed a sync warning.
- The affected file appeared in OneDrive online.
- The latest file version was visible online.
- No additional sync errors were shown.
- The user confirmed that the file was available.

---

## User Communication

**Message to user:**

Hello, the OneDrive sync issue has been resolved. We confirmed that your account was signed in, checked the affected file, restarted OneDrive, and restarted the computer. After the restart, OneDrive synced the file successfully and the warning icon disappeared. Please contact IT support again if the sync warning returns or if other files are affected.

---

## Prevention Notes

To reduce future OneDrive sync issues:

- Keep OneDrive running in the background.
- Avoid unsupported characters in file names.
- Do not shut down the computer while large files are syncing.
- Check the OneDrive icon regularly when working with important files.
- Use OneDrive online to confirm whether a file has uploaded successfully.
- Contact IT support if sync warnings continue.
