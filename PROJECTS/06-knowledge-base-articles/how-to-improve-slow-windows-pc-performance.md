## Purpose

This knowledge base article explains simple steps a user can follow when a Windows computer is running slowly.

The goal is to help users check common performance issues before contacting IT support.

---

## Symptoms

A user may need this guide when:

- The computer starts slowly.
- Applications take a long time to open.
- The computer freezes or becomes unresponsive.
- The fan is loud for a long time.
- Web pages or files open slowly.
- The computer is slower after startup.
- The device feels slower than usual during normal work.

---

## Before You Start

Make sure:

- You save all open work.
- You close applications you are not using.
- You restart the computer if it has not been restarted recently.
- You do not delete files if you are unsure what they are.
- You contact IT support before uninstalling business applications.

---

## Step 1 — Restart the Computer

A restart can fix temporary performance problems.

1. Save all open work.
2. Close open applications.
3. Select **Start**.
4. Select **Power**.
5. Select **Restart**.
6. Sign in again after the restart.

After the restart, check whether the computer is faster.

---

## Step 2 — Close Unused Applications

Too many open applications can slow down the computer.

1. Close applications you are not using.
2. Close browser tabs you no longer need.
3. Wait a few minutes.
4. Check whether the computer responds faster.

If an application is frozen, contact IT support or restart the computer.

---

## Step 3 — Check Startup Apps

Some applications start automatically when Windows starts.

1. Right-click the taskbar.
2. Select **Task Manager**.
3. Select **Startup apps**.
4. Review the list of apps.
5. Disable only apps you recognize and do not need at startup.

Do not disable security tools, backup tools, VPN software, or company-required applications unless IT support tells you to.

---

## Step 4 — Check Available Storage

Low disk space can affect computer performance.

1. Open **Settings**.
2. Select **System**.
3. Select **Storage**.
4. Check the available space on the main drive.
5. Remove only files you no longer need.

You can also empty the Recycle Bin if you are sure the files are no longer needed.

---

## Step 5 — Use Disk Cleanup or Storage Sense

Windows can help remove temporary files.

1. Open **Settings**.
2. Select **System**.
3. Select **Storage**.
4. Review temporary files.
5. Delete only safe temporary files.

Do not delete Downloads, personal files, or work files unless you have checked them carefully.

---

## Step 6 — Check for Windows Updates

Missing updates can sometimes cause performance or stability issues.

1. Open **Settings**.
2. Select **Windows Update**.
3. Select **Check for updates**.
4. Install available updates if allowed by your organization.
5. Restart the computer if Windows asks you to restart.

If your device is managed by your company, follow your organization’s update policy.

---

## Step 7 — Check for High Resource Usage

Task Manager can show which applications are using many system resources.

1. Right-click the taskbar.
2. Select **Task Manager**.
3. Select **Processes**.
4. Check CPU, Memory, and Disk usage.
5. Look for applications using unusually high resources.

Do not end unknown system processes. If you are unsure, take a screenshot and contact IT support.

---

## Step 8 — Check Browser Performance

If the computer is mainly slow while using the internet:

1. Close unused browser tabs.
2. Restart the browser.
3. Try opening the website again.
4. Test another website.
5. Check whether the problem happens only in one browser.

If only one website is slow, the issue may not be the computer.

---

## Step 9 — Check External Devices

Sometimes connected devices can affect performance.

1. Disconnect unnecessary USB devices.
2. Keep keyboard, mouse, headset, and required devices connected.
3. Restart the computer.
4. Check whether performance improves.

Do not disconnect company-required hardware without permission.

---

## Step 10 — Contact IT Support If the Problem Continues

Contact IT support if:

- The computer is still slow after restarting.
- The device freezes often.
- Disk, CPU, or memory usage stays very high.
- Windows updates fail.
- You see error messages.
- The computer becomes slow every day.
- You suspect malware or unusual activity.
- You are not sure which startup apps or files are safe to remove.

When contacting IT support, include:

- When the slowness started.
- Whether the problem happens after startup or all day.
- Which applications are slow.
- Whether other users have the same issue.
- A screenshot of Task Manager if possible.
- Any recent changes, updates, or installations.

---

## User Communication Example

**User message to IT support:**

Hello, my Windows computer is running slowly. The issue started on `<date/time>`. I restarted the computer and closed unused applications, but the problem is still there. It is slow when I use `<application/website>`. I also checked Task Manager and noticed high `<CPU/Memory/Disk>` usage.

---

## Summary

Slow Windows performance can be caused by too many open applications, startup apps, low storage space, missing updates, high resource usage, browser issues, or temporary system problems.

Users should restart the computer, close unused applications, check startup apps, review storage, check for updates, and observe Task Manager.

If the problem continues, IT support should collect performance details, screenshots, recent change information, and affected application names before further troubleshooting.
