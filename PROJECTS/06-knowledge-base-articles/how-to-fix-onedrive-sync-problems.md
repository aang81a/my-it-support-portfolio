## Purpose

This knowledge base article explains simple steps a user can follow when OneDrive files are not syncing correctly.

The goal is to help users check the most common OneDrive sync problems before contacting IT support.

---

## Symptoms

A user may need this guide when:

- OneDrive shows a sync error.
- Files are not uploading or downloading.
- A file is stuck on “sync pending.”
- OneDrive is not signed in.
- Files are missing from the local OneDrive folder.
- The OneDrive cloud icon shows a warning or error.
- Changes made on one device do not appear on another device.

---

## Before You Start

Make sure:

- You are connected to the internet.
- You are signed in with your correct Microsoft 365 work or school account.
- OneDrive is running.
- You have saved your open documents before restarting OneDrive or the computer.

---

## Step 1 — Check the OneDrive Icon

Look for the OneDrive cloud icon in the taskbar notification area.

The icon may show:

- A blue cloud for a work or school account.
- A white cloud for a personal account.
- A red X if there is a sync problem.
- A pause symbol if syncing is paused.
- A spinning symbol if files are currently syncing.

If you do not see the OneDrive icon, click the hidden icons arrow near the clock.

---

## Step 2 — Check If OneDrive Is Signed In

1. Click the OneDrive cloud icon.
2. Check whether OneDrive asks you to sign in.
3. If needed, sign in with your Microsoft 365 work or school account.
4. Wait a few minutes and check whether syncing starts again.

If you are not sure which account to use, contact IT support.

---

## Step 3 — Resume Syncing

Sometimes OneDrive sync is paused.

1. Click the OneDrive cloud icon.
2. Select **Help & Settings**.
3. If you see **Resume syncing**, click it.
4. Wait and check whether the files start syncing.

---

## Step 4 — Check the File or Folder Name

OneDrive may not sync files with unsupported names or characters.

Check whether the file name contains special characters such as:

- `"`
- `*`
- `:`
- `<`
- `>`
- `?`
- `/`
- `\`
- `|`

Rename the file if needed and try syncing again.

---

## Step 5 — Check File Size and Storage Space

OneDrive sync may fail if there is not enough storage space.

Check:

- Whether your computer has enough free disk space.
- Whether your OneDrive storage is full.
- Whether the file is very large.

If storage is full, delete unnecessary files or contact IT support.

---

## Step 6 — Restart OneDrive

1. Click the OneDrive cloud icon.
2. Select **Help & Settings**.
3. Select **Quit OneDrive**.
4. Open the Start menu.
5. Search for **OneDrive**.
6. Open OneDrive again.

Wait a few minutes and check whether the sync problem is resolved.

---

## Step 7 — Restart the Computer

If OneDrive is still not syncing:

1. Save your work.
2. Restart the computer.
3. Sign in again.
4. Check the OneDrive icon.

A restart can fix temporary sync or sign-in problems.

---

## Step 8 — Check OneDrive Online

1. Open your browser.
2. Go to OneDrive online.
3. Sign in with your Microsoft 365 account.
4. Check whether the missing file is visible online.

If the file is visible online but not on the computer, the issue is likely local sync.

If the file is missing online too, it may not have uploaded successfully.

---

## When to Contact IT Support

Contact IT support if:

- OneDrive still shows a red X after the steps above.
- You cannot sign in.
- Important work files are missing.
- OneDrive asks for repeated sign-in or MFA approval.
- You receive an error message you do not understand.
- The same sync problem keeps returning.

When contacting IT support, include:

- A screenshot of the OneDrive error.
- The file or folder name.
- When the problem started.
- Whether the file appears in OneDrive online.
- Whether other files are syncing correctly.

---

## User Communication Example

**User message to IT support:**

Hello, my OneDrive is not syncing correctly. The issue started today. I checked the OneDrive icon and it shows a sync error. I restarted OneDrive and the computer, but the problem is still there. The affected file is `<file name>`. I also checked OneDrive online and the file is / is not visible there.

---

## Summary

OneDrive sync problems are often caused by sign-in issues, paused syncing, unsupported file names, storage limits, or temporary app problems.

Users should first check the OneDrive icon, confirm they are signed in, resume syncing if needed, restart OneDrive, and verify whether the file appears online.

If the issue continues, IT support should collect the error message, affected file name, account information, and sync status before further troubleshooting.
