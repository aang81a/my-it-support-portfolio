**Best 3 diagrams to start with**

Do these first:

Lab environment overview

Active Directory structure

Permissions / shared folder access

**That is enough for a very good beginner portfolio project.**

.

**Tools you can use**

The easiest options are:

draw.io / diagrams.net

PowerPoint

Canva

even a clean hand-made box diagram in Word or Google Docs exported as image

For a GitHub portfolio, simple is better than fancy.

.


Keep them simple

Use:

boxes

arrows

short labels

Do not try to make them look too technical or crowded.
