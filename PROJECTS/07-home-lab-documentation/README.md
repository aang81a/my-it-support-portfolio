# Home Lab Documentation

## Overview
This project documents a virtualized IT support home lab built to practice core help desk and system administration tasks in a controlled environment.

## Lab Environment
- Virtual machine platform: VirtualBox / VMware
- Windows Server
- Windows client
- Active Directory test environment

## Objectives
- Build a virtualized Active Directory lab
- Join a client machine to a domain
- Create and manage user accounts
- Assign permissions
- Test shared folder access
- Document the setup clearly

## Lab Components
- Domain controller
- Client workstation
- Active Directory users and groups
- Shared folders and permissions

## Tasks Completed
- Created virtual machines for server and client systems
- Configured a basic lab network
- Installed and configured Active Directory
- Joined a client machine to the domain
- Created user accounts and groups
- Assigned and tested folder permissions
- Verified shared folder access from a client machine

## Skills Demonstrated
- Windows support
- Active Directory fundamentals
- User account management
- Domain join
- Permissions management
- Shared folder access
- Technical documentation
