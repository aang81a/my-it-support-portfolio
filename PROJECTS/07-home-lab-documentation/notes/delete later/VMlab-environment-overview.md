# Lab Environment Overview

## Goal
Create the two virtual machines needed for the home lab.

## What you need
- VirtualBox or VMware
- Windows Server ISO
- Windows 10 or 11 ISO
- Enough RAM and disk space for two virtual machines

## Steps
1. Install VirtualBox or VMware on your computer.
2. Create the first virtual machine and name it `DC1`.
3. Install Windows Server on `DC1`.
4. Create the second virtual machine and name it `CLIENT1`.
5. Install Windows 10 or Windows 11 on `CLIENT1`.
6. Configure both virtual machines to use the same lab network.
7. Start both machines and confirm they can run normally.

## Expected result
You have two working virtual machines:
- `DC1` for the server
- `CLIENT1` for the client

## Notes
This is the base of the lab. Do not continue until both virtual machines are created and working.
