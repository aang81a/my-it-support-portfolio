# Permissions Test

## Objective
Test whether assigned users can access shared folders based on their permissions.

## Steps
1. Created a shared folder on the server
2. Assigned permissions to a user group
3. Logged in from the client machine
4. Verified folder access
5. Confirmed that unauthorized access was blocked where expected

## Result
Shared folder access matched the assigned permissions and confirmed the basic access control setup.
