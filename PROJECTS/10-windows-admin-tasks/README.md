# Windows Admin Tasks

## Overview
This project documents basic Windows administration tasks relevant for IT Support and Help Desk roles.

## Objective
Practice common administrative tasks that are often handled in first-level or junior IT support environments.

## Tools
- Windows 11
- Windows Server / Active Directory lab
- Control Panel / Settings
- Command Prompt / PowerShell basics

## Practice Tasks
- Create and manage local users
- Check system information
- Review installed applications
- Manage Windows updates
- Check device manager
- Review network configuration
- Use basic command-line tools
- Manage shared folder access
- Document common support steps

## Example Commands
```cmd
ipconfig
ping
nslookup
whoami
hostname
tracert
```

## Skills Demonstrated
- Windows support
- Basic system administration
- User account tasks
- Basic networking checks
- Command-line fundamentals
- Documentation
