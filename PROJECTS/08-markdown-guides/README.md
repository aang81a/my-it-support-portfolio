## Overview

This section contains short Markdown-based quick-reference guides for common IT support tasks.

The guides are intentionally concise and focus on the core workflow of each task: purpose, steps, expected result, and important support notes.

---

## Purpose

The purpose of this section is to demonstrate clear technical documentation in Markdown format.

Unlike the full knowledge base articles, these guides are not written as polished end-user articles. They are short internal support notes that show how common procedures can be documented in a simple, structured, and consistent format.
