# How to Join a PC to a Domain

## Purpose
This guide explains how to connect a Windows computer to a domain.

## Steps
1. Open system settings on the client computer.
2. Verify that the DNS server points to the domain controller.
3. Open the computer name and domain settings.
4. Select the option to join a domain.
5. Enter the domain name.
6. Provide domain administrator credentials when prompted.
7. Restart the computer.

## Result
The computer becomes part of the domain and can be used with domain user accounts.

## Notes
Network connectivity and correct DNS settings are required for a successful domain join.
