# How to Create a User in Active Directory

---

## Purpose

This guide explains how to create a new user account in Active Directory.

It is intended as a short internal support note for IT support tasks.

---

## When to Use

Use this guide when:

- a new employee or test user account needs to be created
- a user account must be added to the correct organizational unit
- a lab environment requires new domain users
- IT Support needs to prepare an account before assigning access

---

## Steps

1. Open **Active Directory Users and Computers**.
2. Navigate to the correct organizational unit.
3. Right-click inside the organizational unit and choose **New > User**.
4. Enter the required user details, such as first name, last name, and logon name.
5. Set an initial password according to company password policy.
6. Configure password and account options, such as **User must change password at next logon**.
7. Save the account.
8. Verify that the user account appears in the correct organizational unit.
9. Add the user to required security groups if approved.
10. Confirm that the account is ready for login or further access assignment.

---

## Result

A new Active Directory user account is created in the correct organizational unit.

The account is ready for login testing or access assignment based on the user’s role.

---

## Notes

User accounts should follow company naming conventions and account creation procedures.

Group membership and permissions should be assigned based on the user’s role and approved access requirements.

Do not assign administrative rights unless this is approved and required.

Document the created account, assigned groups, and any access changes according to company policy.
