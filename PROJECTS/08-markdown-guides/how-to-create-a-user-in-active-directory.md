# How to Create a User in Active Directory

## Purpose
This guide explains how to create a new user account in Active Directory.

## Steps
1. Open Active Directory Users and Computers.
2. Navigate to the correct organizational unit.
3. Right-click and choose **New User**.
4. Enter the required user details.
5. Set an initial password.
6. Configure password and account options.
7. Save the account and verify that it appears in the correct location.

## Result
A new user account is created and ready for access assignment.

## Notes
Group membership and permissions should be assigned based on the user’s role.
