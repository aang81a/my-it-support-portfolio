# Network Internet Access Ticket

## Overview

This ticket documents a simulated first-level IT support case in Spiceworks.

The user reported that the computer was connected to Wi-Fi, but websites did not load. The ticket was categorized as a network issue, assigned high priority, documented with troubleshooting notes, and closed after the issue was resolved.

## Problem

User reports that the computer is connected to Wi-Fi, but websites do not load.

## Analysis

The issue was treated as a network connectivity problem.

The support process focused on checking whether the issue affected only one user or multiple users, confirming Wi-Fi connection status, checking IP configuration, and testing DNS/connectivity.

## Troubleshooting Steps

- Confirmed whether the issue affects only one user or multiple users.
- Checked Wi-Fi connection status.
- Asked user to test another website.
- Checked IP configuration.
- Used ping test to check connectivity.
- Checked DNS resolution.
- Restarted network connection.

## Resolution

Issue resolved after reconnecting to Wi-Fi and confirming DNS/connectivity tests.

## Result

User confirmed that internet access was restored and websites loaded again.

Final ticket status: **Closed**
