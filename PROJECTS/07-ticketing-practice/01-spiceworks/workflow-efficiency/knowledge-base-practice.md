# Spiceworks Knowledge Base Practice

## Purpose

This document shows how existing support articles can be connected to ticketing practice in Spiceworks.

## Practice Idea

Existing knowledge base articles can be used as reusable support references when handling repeated user issues.

## Example Use

| Ticket Scenario | Related KB Article | How it supports the ticket |
|---|---|---|
| Browser-based application does not open | Browser troubleshooting article | Provides user-facing steps before or after ticket resolution |
| Network / internet access issue | Basic internet troubleshooting article | Supports first-level checks before escalation |
| Shared folder access denied | Shared folder access article | Helps explain access checks and escalation need |

## Skills Practiced

- Connecting tickets with reusable support documentation
- Using KB articles to reduce repeated explanations
- Supporting users with consistent step-by-step guidance
- Separating ticket notes from user-facing instructions
